# **App Name**: Button Box

## Core Features:

- Basic Button Display: Render a visually distinct button element on the screen with a generic label.
- Click Interaction Feedback: Provide immediate visual feedback (e.g., subtle color change or press effect) when the button is clicked.

## Style Guidelines:

- Primary color: A clear, stable blue (#2BABDB) to convey reliability and interactivity.
- Background color: A very light, desaturated blue (#EBF4F8) that provides a clean, open canvas without distracting from the button.
- Accent color: A vibrant greenish-cyan (#13AD93) for subtle highlights or interaction states, providing an energetic contrast.
- Body and headline font: 'Inter', a neutral and modern sans-serif, ensuring readability for button labels.
- A minimalistic, centered layout to highlight the single button as the primary UI element.
- Subtle hover and press animations on the button to enhance its responsiveness and interactivity.
